1: Introduction

Tutorial level: Advanced

Requirements: You should be familiar with using the main functions of Help & Manual before working through this tutorial. Among other things, you should know how to create and edit projects and be able to publish your projects to different output formats.

This tutorial project walks you through using the conditional output capabilities of Help & Manual. You will learn how to use build conditions to control the content included in your output. The tutorial covers format-based conditions and user-defined conditions and shows you how to use them to include or exclude topics, chapters and topic content.

For full details on Help & Manual's conditional output features see the following topics in the help:

•More Advanced Procedures > Conditions and Customized Output

•Reference > Modules, Conditional Output and Variables

How to use this sample project:

1.Open this project file (CondOutput.hmxz) and publish a HTML HELP/CHM file. In the Publish dialog leave the default HTML HELP Include Option selected and make sure that all other options are unselected. Then view the  published help file and compare it to the project here in Help & Manual.

2.Publish the project again, but this time also select Demo Build, in addition to HTML Help. Compare the result with the project here in Help & Manual again. In particular, compare the first five numbered chapters.
