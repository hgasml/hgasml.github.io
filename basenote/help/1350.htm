Меню "Севис" - Параметры

Форма "Параметры"

Меню Сервис- Команда - "Параметры…".

 

Параметры – «Просмотр»

 

При сворачивании программы она прячется, и появляется икона на панели часов ОС. Двойной клик мышью показывает программу снова. Правой клавишей мыши вызываются дополнительные команды.

Показывать/прятать пункт меню «Редактор базы»

Показывать/прятать пункт меню «Редактор данных»

Показывать/прятать страницы представления информации.

В формах представления данных есть поле «ЦЕНА». Его отображение может быть изменено этими параметрами через создание формулы.

Икона программы

 

 

Есть возможность определить вид иконы программы при сворачивании ее в системный Try.

 

 

Параметры – "Поиск"

 

Определяет поле, в котором будет производиться поиск, в случае поиска по одному полю.

Поля комплексного поиска

Определяются поля, в которых будет производиться поиск, в случае комплексного поиска.

 

Параметры – «Настройки»

Авто замена в директории документа

 

Определяются параметры дополнительного поиска файлов, через замену символов в пути файла. В данном окне можно задать список строк замены. Например, можно заменять директории. Новый путь файла по авто замене в базу данных не записывается.

Данный флаг применяйте при создании лазерных инсталляций или при перезаписывании (переносе) базы и ее файлов на другой логический диск, создание резервной копии.

Определяются дополнительные директории поиска файлов.

1.Отмеченная - поиск производится в директории и во всех поддиректориях.

2.Серая - поиск производится только в директории.

3.Не отмеченная - поиск отключен.

Таким образом, зарегистрированные файлы разрешается перемещать в другие директории с сохранением ИМЕНИ файла.
