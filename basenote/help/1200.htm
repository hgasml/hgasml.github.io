Меню "Просмотр"

 

Команды «Фильтр по дате и времени» - в данной группе можно выбрать глобальный фильтр по времени регистрации информационного ресурса. Фильтр является общим для всех информационных записей. Вид фильтра отображается в нижней части программы на панели подсказок. Фильтр удобно применять при большом количестве информационных записей.

Команда - "Скрытые записи" – разрешает просмотр записей, у которых параметр Видимость не отмечен.

Команда - "Просмотр полей «О чем»" – разрешает просмотр поля «Описание». Применяйте эту опцию, если описание содержит много текста.

Команда - "Копия формы «О Чем»" – создается отдельная форма с описанием информации. Из этой формы можно быстро находить запись, редактировать описание.

Команда - "Открыть файл" - просмотр файла, связанного с информационной записью.

Команда - "Открыть файл как..." - запускается на просмотр форма с программами просмотра файлов. Выберите программу для чтения файла.

Команда - "Просмотр..." – открывается специализированная форма, в которой просматривается файл, связанный с информационной записью. Форма располагается поверх и не мешает работе программы.

Команда - "Проводник" - открывается окно проводника WINDOWS с директорией выбранного файла информационного ресурса.

Команда - "Свойства файла…" - открывается окно, показывающее свойства файла. Из формы доступны команды операционной системы для выбранного файла (открыть, копировать, вставить и т.д.). Форма свойств располагается поверх форм и не мешает работе программы.

Вкладка «Таблица» - на данной вкладке расположена группа команд предназначенных для сохранения и выставления настроек таблицы отображения информации. Настройки можно сохранять и читать из файла. Настройте вид одной таблицы и распространите его через память обмена компьютера. Настройки таблицы сохраняются при закрытии базы данных.
