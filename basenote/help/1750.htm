Форма закладок

 

Две команды и индикатор. Сохранить закладки, индикатор выполнения действий, прочитать закладки. При закрытии программы закладки сохраняются автоматически.

 

Создать новую закладку. Удалить текущую закладку со всеми записями на ней.

 

Добавить файл, директорию. Удалить выделенные записи.

Добавить документ в закладку из базы данных можно через команду меню «Сервис»-Документ в закладку.

 

Дополнительные команды

 

Свернуть, развернуть форму.

 

Кроме того, правой клавишей мыши на записи в закладке – доступны дополнительные команды.
