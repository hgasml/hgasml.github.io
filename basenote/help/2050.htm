Дополнительные команды календаря

Напоминание - открывается форма, всех напоминаний для данного календаря. В форме напоминаний можно управлять, редактировать и находить события.

Печать – печать отображаемой информации о планах.

Экспорт – команды для сохранения событий в разные текстовые форматы.

Экспорт – Таблица, файл обмена – открывается вид просмотра событий в табличной форме. Через табличную форму можно выделять, находить, экспортировать выделенные события в отдельный файл. Этот Файл открывается командой «Импорт…» на отдельной странице планирования.

 

Изменение шкалы времени дня: Правая клавиша мыши на шкалу времени.

 

- для таблицы событий устанавливается временной фильтр Начало-Конец. Параметры времени берутся из календаря просмотра.

- выделяются все записи в таблице

- при включенном состоянии выбранное событие в таблице автоматически находится в календаре.

- выделенные события сохраняются в отдельный файл – файл экспорта планов. В этом файле отсутствует личная информация пользователя.

- Открывается файл экспорта планов на отдельной вкладке.

- поиск слова в таблице планирования

- команда Переместить выбранные события в другой плановый ресурс

- команда Скопировать выбранные события в другой плановый ресурс

- команда Удалить выбранные события

- Обновить события плана

 

Планы отображаются на вкладках. На вкладках доступна кнопка «закрыть».

При нажатии на кнопку закрыть предлагается выполнить команды по отображению формы планирования.

Если выбрать команду:

«Спрятать план, закрыть вкладку» - Вкладка с планом закроется, в таблице "Планирования" для данного плана установится флаг скрытости для отображения во вкладках.

«План как отдельная форма ...» - План откроется в отдельной форме. При закрытии этой формы вкладка восстановится.

«Закрыть форму диалога.» - Никаких действий не предпринимается.
