Поиск информации

На странице представлен интерфейс для "поиска информации".

Поиск производится по ключевому (ключевым) словам в выбранных Вами информационных полях.

 

Команды поиски доступны как из пунктов меню, так и из панели команд формы поиска.

 

 

Команды Меню «Поиск»

открывается форма ввода ключевого слова для поиска в базе данных (смотрите ниже описание формы)

открывается форма ввода нескольких ключевых слов для поиска в базе данных по заданному логическому алгоритму

становится активной вкладка «Поиск информации»

Создается новая вкладка, в которой можно использовать поисковый запрос

В таблице отображающей, информацию показывается панель для ввода дополнительного фильтра информации.

Таблица поиска информации

Команда выделяет все записи в таблице, где отображается найденная информация. Таблица располагается в нижней части поисковой формы.

Таблица поиска информации

Команда копирует выделенные записи в таблице, где отображается найденная информация в память в текстовом формате.

Таблица поиска информации

Команда сохраняет в файл выделенные записи в таблице, где отображается найденная информация в текстовом формате.

Таблица поиска информации

Команда сохраняет в файл выделенные записи в таблице, где отображается найденная информация в формате электронных таблиц.

Таблица поиска информации

Команда сохраняет в файл выделенные записи в таблице, где отображается найденная информация в формате интернета.

Таблица поиска информации

Команда сохраняет в файл выделенные записи в таблице, где отображается найденная информация в формате электронного документа.

Таблица поиска информации

Открывается форма при, помощи которой, можно автоматически зарегистрировать группу файлов используя один шаблон заполнения информацией.

Таблица поиска информации

В качестве ключевого поля используется имя файла. Поиск производится по специальному алгоритму.

Таблица поиска информации

Производится поиск в рубрикаторах которые располагаются на страницах В и Г

 

Команды формы поиска

открывается форма ввода ключевого слова для поиска в базе данных (смотрите ниже описание формы)

открывается форма ввода нескольких ключевых слов для поиска в базе данных по заданному логическому алгоритму

Программа откроет файл связанный с информационным ресурсом (если он есть)

Программа предложит список программ для открытия файла, связанного с информационным ресурсом (если он есть)

Будет выполнен поиск активной информационной записи на СТРАНИЦЕ А

Меню определяет выбор, какое слово будет искаться в текстовом описании ресурса. Несколько слов появляется, если поиск производится с разделением слов.

Дополнительные команды для выбранного ресурса.

Если открыта форма личных закладок, информационный ресурс запишется как закладка.

 

 

Форма поиска информации одной строкой.

Фраза поиска -

Определяются слова для поиска. Параметры поиска определяются в форме "Параметры" и флагами ниже.

Определить поиск в одном поле или группы полей информационной записи. Вызвать форму Параметры-поиск.

Будет искаться точное совпадение фразы.

Будет искаться обязательное вхождение каждого слова из фразы.

Будет искаться вхождение слова из фразы (любого).
