Описание задачи

Для задачи определяются:

•Краткое описание.

•Присоединять файлы.

•Дата завершения задачи.

•Дата начала задачи.

•Статус по выполнению.

•Важность по рейтингу.

•Процент исполнительности

•Время напоминания о задаче.

•Метка о владельце задачи.

•Файл, который будет исполнен по времени напоминания.

•Более подробное описание в тестовом виде.

•Список присоединенных файлов.

•Атрибуты цвета и шрифта задачи в списке задач.
