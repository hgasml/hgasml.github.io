Обновление до новой версии

Выполните команду меню ?-Обновление.

В форме "Обновление" проверьте наличие новых версий ПО. Выполните команду "Сайт" для перехода на страницу с инсталляциями. Можно скачать архив и из архива обновить ваше ПО.

 

Обновление до новой версии баз данных.

Возможно в будущем возникнет необходимость обновить структуру БАЗЫ ДАННЫХ. Для этого скачивается папка NewVer с файлами. Файл NewVer.bdf - содержит новую структуру базы данных.

Программа NewVer.exe легка в использовании.

Запустите NewVer.exe, определите ваш файл базы данных для обновления, выполните команду "Выполнить сравнение баз данных". После этой команды происходит сравнение с дистрибутивом. Не совпадения отображаются в программе.

 

Выполните команду «Синхронизировать все таблицы выбранной базы данных». Происходит обновление до новой версии. Выполните повторно команду «Выполнить сравнение баз данных» и убедиться в правильном обновлении.
