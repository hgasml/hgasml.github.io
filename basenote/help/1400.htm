Меню "Редактор базы"

 

 

Команда - "Каталог" - формирование древовидного рубрикатора. Каталог является дополнительным рубрикатором информационных записей.

Команда - "Планы…" – формирование модели планирования.

Команда - "Задачи…" – формирование  списков задач.

Команда - "Рубрики…" - открывается форма редактирования рубрик. Рубрики применяются при описании информации в базе данных.

Команда - "Место хранения…" - открывается форма по редактированию мест хранения информации. Применяется при описании информации в базе данных.

Команда - "Программы просмотра…" - открывается форма редактирования программ, с помощью которых можно просматривать файлы. Данная форма является расширением просмотра файлов.

Команда - "Тип файла…" - открывается форма редактирования типов файла. Информационной записи можно определять тип.

Важно: При регистрации "директории" обязательно задайте тип "Директория". Тип "Информация" не связан с файлом и является информацией.

Команда - "Шаблоны…" - открывается форма редактирования текстовых шаблонов. При помощи шаблонов можно создавать сразу готовое текстовое поле, заполнять поля значениями из базы данных.

 

Адресная книга

В программе предусмотрена как отдельное направление – кадровая задача, описывающая владельцев информационных ресурсов. Информация о владельце состоит из краткого описания информации о владельце, должности, организации владельца.

Команда - "Должности …" - открывается форма редактирования должностей владельцев информации. Должности применяются в таблице владельцев информации.

Команда - "Организации …" - открывается форма редактирования организации владельцев информации. Организация применяется в таблице владельцев информации.

Команда - "Адреса …" - открывается форма редактирования владельцев информации. Фактически это кадровая задача.

 

Изображения

Команда - "Свойства базы данных" - открывается форма редактирования краткого описания, свойств и назначение базы данных.

Команда - "Этикетки …" - открывается форма редактирования графических этикеток. Этикетка представляет собой графический файл малого размера. Рекомендуются размеры графического файла 32x32 пикселя. Применяется при регистрации нового документа.

Команда - "Реклама …" - открывается форма редактирования рекламных роликов.

 

Пользователь

Понятие пользователя применимо к текущему сеансу работы в программе. Если определен Пользователь то доступны дополнительные сервисы:

•работа с формой закладок.

 

Команда - "Пользователи …" - открывается форма редактирования пользователей для базы данных. Пользователи применяются при открытии формы закладок. У каждого пользователя свои закладки.
