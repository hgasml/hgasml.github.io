Меню "Поиск"

Меню «Поиск» обслуживает интерфейс, представленный на вкладке «Поиск информации».

 

 

 

Команда - "Поиск по одному ключу" - открывается форма ввода слов для поиска в базе данных.

 

Команда - "Поиск по ключевым словам" - открывает форму для поиска в базе данных нескольких слов по заданному логическому алгоритму. Предусмотрено ввод до 8 ключевых слов.

 

Команда - "Раздел поиска" - открывается вкладка с последними значениями найденной информации - становится активной вкладка «Поиск информации».

 

Команда - "Создать форму поиска" – Создается новая вкладка для поиска.

 

Команда - "Вид фильтра" - в текущей таблице открывается панель для построения фильтра в таблице информации. Смотрите рисунок ниже.

 

Команда - "Выделить все" - в таблице найденной информации выделяются все записи.

 

Команды - "таблица найденной информации" скопировать в память и экспорт в файл, относятся к таблице найденной информации, выделенные записи.

 

Команда - "Поиск не учтенных файлов" - открывается окно выбора файлов. Выбранные файлы проверяются на регистрацию в базе данных. Результат проверки отражается в специальной форме регистрации. Окно выбора файлов можно закрыть. В этом случае откроется форма регистрации документов. Для проверки регистрации файлов просто перетащите файлы на форму. Двойной щелчок мыши на записи файла вызывает форму добавления нового документа.

 

Команда - "Поиск по файлу" - открывает окно для выбора файла и поиск выбранного файла в базе данных. В качестве ключевого слова поиска используется имя файла. Поиск производится по специальному алгоритму.
