Редактирование справочников в одной таблице

Редактирование справочников (данных)

Для правильной работы программы необходимо заполнять информационные справочники. Создание справочников производится через пункт меню «Редактор базы». Формирование таблиц справочников производится одинаковым способом. Рассмотрим редактор на примере одного справочника – «Таблица Рубрик».

 

 

Назначение кнопок для работы с таблицей следующее:

- - добавить новую запись в таблицу;

- - изменить существующую запись в таблице;

- - удалить существующую запись из таблицы;

- - отсортировать записи в таблице по имеющимся полям в порядке возрастания или убывания;

- - данные кнопки служат для закрытия формы.

- - данная кнопка, если доступна, возвращает выбранное значение из справочника.
