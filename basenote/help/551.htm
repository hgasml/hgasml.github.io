Работа с таблицей данных.

Правая клавиша мыши на колонку вызывает контекстное меню с дополнительными командами.

 

 

1.Сортировка данных. Удерживая клавишу SHIFT, происходит сортировка по нескольким столбцам.

2.Группировка по полям. Поля можно перетаскивать на панель группировки. При этом происходит трансформация в отображении информации по древовидному принципу.

3.Нижняя сноска предоставляет возможность выполнять агрегатные функции ко всем записям колонки.

4.Скрывать/отображать колонки. Менять их размер и расположение. Выставлять оптимальный размер выбранной колонки, всех колонок.

5.Формировать фильтры любой сложности.

6.Все настройки автоматически сохраняются в базе данных.

 

Настройки таблицы можно сохранять в файл, читать из файла, копировать в память, применять из памяти к другой таблице.
