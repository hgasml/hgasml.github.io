Введение

Управление знаниями, планирование, записная книжка.

 

Программа обеспечивает удобную работу с информацией, которая обычно храниться в наших записных книжках (адреса, телефоны, дни рождения, задачи, напоминания, ссылки, файлы).

Программа предоставляет средство создания базы данных, которая связана с файлами. Таким образом, база данных может состоять из информационных ссылок на файлы. В этих ссылках содержится информация о файле и его характеристиках. Эти характеристики определяются пользователем. Например: такая информация о файле как: тип файла, краткое описание, о чем информация, название темы информации, раздел в рубрике и т.д.

 

Программа позволяет удобно работать с информацией:

•добавлять, редактировать, удалять, производить поиск и рубрицирование по различным критериям;

•формировать адресную книгу. Производить поиск и рубрицирование по различным критериям;

•формировать многоуровневое планирование. Производить поиск,  сортировку по различным критериям;

•формировать списки задач.

 

Таким образом, чем полней будет описание, тем легче найти нужную вам информацию. Во время просмотра программа сама автоматически определяет тип файла и показывает его в нужной форме. Если это графический файл, то вы увидите изображение, если файл звуковой то вы услышите звук. Поиск информации осуществляется при помощи текстовых запросов на естественном языке.

Минимальные требования:

Процессор – Pentium 2,6 ГГц.

Память – 32 M.

Свободное место HDD – 130 Mb (зависит от размера Ваших баз данных).
